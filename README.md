# 🚀 AI/ML Projects Portfolio

This repository showcases a collection of end-to-end AI/ML projects developed during my internship. Each project focuses on solving real-world problems using machine learning and is deployed with an interactive Streamlit interface.

---

## 📌 Projects Overview

### 📰 1. Fake News Detection

* **Objective:** Classify news articles as Fake or Real
* **Tech:** NLP, TF-IDF, Logistic Regression
* **Dataset:** https://www.kaggle.com/datasets/saurabhshahane/fake-news-classification
* **Highlights:**

  * Combined title + content for better context
  * Text preprocessing and vectorization
  * Real-time prediction using Streamlit

---

### 📄 2. AI Resume Ranker

* **Objective:** Rank resumes based on job description similarity
* **Tech:** TF-IDF, Cosine Similarity
* **Dataset:** https://www.kaggle.com/datasets/snehaanbhawal/resume-dataset
* **Highlights:**

  * Automated resume screening
  * Similarity-based ranking system
  * Efficient recruitment support tool

---

### 🎬 3. Movie Recommendation System

* **Objective:** Recommend movies based on user preferences
* **Tech:** TF-IDF, Cosine Similarity
* **Dataset:** TMDB Movies Dataset
* **Highlights:**

  * Content-based filtering
  * Partial matching for better search
  * Streamlit UI with posters and platform info

---

### 💳 4. Fraud Detection System

* **Objective:** Detect fraudulent transactions
* **Tech:** Isolation Forest (Anomaly Detection)
* **Dataset:** https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud
* **Highlights:**

  * Handles highly imbalanced data
  * Detects anomalies in transaction patterns
  * Real-time prediction interface

---

### ✍️ 5. AI Text Summarizer

* **Objective:** Generate concise summaries from long text
* **Tech:** Hugging Face Transformers (BART/T5)
* **Dataset:** Not required (pretrained models)
* **Highlights:**

  * Transformer-based summarization
  * Handles large input text
  * Fast and accurate output

---
